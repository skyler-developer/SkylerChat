# Skyler-Chat

## 本项目旨在为用户提供智能问答服务。

## 技术栈
1. 框架：Next.js
2. 包管理：npm
3. 状态管理：zustand
4. 接口：Next.js API能力

## 开发
1. 初始化：npm i
2. 启动：npm run dev

### 功能点
1. 用户登陆注册
   - 采用...加密算法
2. 智能体创建
3. 聊天
   - 错误重试机制
4. 翻译
5. 导出md
6. 使用mysql2，无需手动释放连接池
